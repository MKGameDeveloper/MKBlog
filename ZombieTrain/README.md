# 🚂 좀비 트레인 (Zombie Train)

끝없이 연결된 선로를 달리는 기차에서 살아남는 좀비 서바이벌 게임.
**탑승 인원 한 명 한 명의 상태**(체력·포만·수분·멘탈·면역·체온)를 살피며, 식량·물·연료·기차 내구도를 관리하고,
도시·평야를 2.5D로 탐험해 **건물에 들어가 가구를 뒤지며** 물자를 모으고, 생존자를 구출해 동료로 삼으세요.

순수 HTML/CSS/JavaScript + Canvas. 빌드 도구·외부 라이브러리·에셋 없음.

## 실행 방법

`index.html` 을 브라우저로 열기만 하면 됩니다. (Chrome/Edge/Firefox 최신 버전 권장)
진행은 `localStorage` 에 자동 저장. 우상단 `↺` 로 새 게임.

## 조작

| 키보드 | 터치 | 동작 |
|---|---|---|
| ◀▶▲▼ / WASD | D-pad | 이동 (탐험은 일부러 느립니다) |
| Shift (이동 중) | — | 달리기 — 스태미나 소모 · **소리가 멀리 퍼져 좀비를 부름** |
| E | ✋ 버튼 | 상호작용 — 가구 뒤지기(약탈창) · 🔒문 부수기/열기·닫기 · 구출 · 농사 · 🚂적재 |
| **마우스 클릭** | 화면 터치 | **공격 — 클릭한 지점을 향해 부채꼴로 휘두름** (탐험 전용) |

> 탐험에서 **공격(클릭)과 상호작용(E)은 분리**되어 있습니다 — 가구를 뒤지려다 실수로 좀비를 때리는 일을 막기 위함입니다. 공격은 **클릭한 위치를 향해 부채꼴(사거리=반지름, 폭=각도)로** 휘두르며, 바닥에 사거리·반경 안내선이 보입니다. 무기를 장착하면 반지름과 각도가 넓어집니다. 기차 안에서는 공격이 없으므로 가까운 칸에서 E로 상호작용합니다.
| 하단 메뉴 | 하단 메뉴 | 운행 · 지도 · 농사 · 동료 · 정비 · 증설 · 도움말 |

## 화면 구성

- **상단 바**: 🍞식량 · 💧물 · ⛽연료 · 🛠️내구도 (비축/기차 자원) · 🌦️날씨
- **우측 크루 카드 (항상 표시)**: 인원별 ❤️체력 · 🍖포만 · 💧수분 · 🧠멘탈 · 🛡️면역 · 🌡️체온 — 탭에 들어가지 않아도 항시 추적. 감염·저체온이 진행되면 카드 옆에 💬경고 말풍선이 가끔 뜹니다.
- **기차 모드(횡스크롤)**: 도시→산→강→역→평야… 바이옴이 실시간으로 흘러갑니다
- **탐험 모드(2.5D 탑뷰)**: 넓은 맵을 측면으로 기울인 오블리크 뷰. 건물이 입체로 돌출하고, **출입문으로 들어가면 지붕이 반투명**해지며 내부 가구가 보입니다. 좌상단 **🗺️미니맵**에 전체 맵·플레이어·건물(잠김=붉은색)·좀비·기차·현재 화면 범위가 표시됩니다.

## 핵심 루프

1. **주행** — 연료가 계속 소모(보통속도 실시간 ~10분). 비축 식량/물에서 인원들이 자동으로 먹고 마십니다.
   - 식량/물 비축이 바닥나면 개인 🍖포만/💧수분이 떨어지고 → ❤️체력 감소.
   - ⛽연료 0 → 불모지에 강제 정차.
2. **지도(🗺️)에서 정차 계획** — 다가오는 정차지와 📍외부 농장 핀을 보고 **정차 목표를 설정**하면 도착 시 자동 정차.
3. **정차 → 탐험(2.5D)** — 천천히 이동하며:
   - 🌳은신처에 들어가거나 **건물 벽 뒤로 시야를 차단**해 좀비를 회피 (좀비는 벽에 가리면 못 보고, 쫓기다 시야를 잃으면 마지막 목격 지점까지만 왔다가 포기)
   - 🏠**약탈(좀보이드식)**: 건물 **출입문으로 들어가** 🧊냉장고·🗄️캐비넷·📒책상·🧰서랍장·🛒진열장에 **[E]로 다가가 뒤지면(1~2초)** **약탈 인벤토리**가 열립니다 — 왼쪽=수납공간, 오른쪽=내 가방. 항목을 **담기**로 가져오세요(가구마다 물자 경향이 다르고 건물 색깔과 합쳐짐). 자동차 트렁크도 동일.
   - 🎒**인벤토리·무게**: 휴대에는 **무게 한도**가 있습니다. 👜가방을 주워 **장착**하면 한도가 늘고, 🧥옷을 장착하면 **보온**됩니다(🥵폭염엔 두꺼운 옷이 오히려 위험). 가방이 차면 🚂기차 앞에서 **[E] 적재**로 자원을 비축으로 옮겨 무게를 비우세요(복귀 시 자동 적재).
   - ⚔️**무기 줍기**: 약탈 중 낮은 확률로 무기를 발견합니다(내구도 20~60% 닳은 상태). 약탈창에서 담으면 맨손일 때 자동 장착.
   - 🛠️**자동차 알람(기계전문가)**: 자동차에서 **[E]**로 부품을 써 알람을 설치하고, 다시 **[E]**로 작동시키면 **매우 넓은 반경**의 좀비(추격 중인 좀비 포함)가 그 차로 몰려갑니다 — 멀리 떨어진 차에 켜서 좀비를 따돌리는 전술용.
   - 🔒**잠긴 문 / 문 여닫기**: 건물은 **높은 확률로 잠겨** 있습니다. 문 앞 **[E]**로 **부수거나**(시간↑·**큰 소리**) **🔧제작전문가**가 있으면 **조용히·빠르게 따고** 들어갑니다. 들어간 뒤 **[E]로 문을 닫으면** 좀비는 **부수기 전까진 못 들어옵니다**(좀비는 닫힌 문을 시간을 들여 부숨).
   - 🔊**소리·청각 AI**: **달리기·공격·문 부수기**는 소리가 퍼지고, 좀비는 **시야에 없어도 소리가 들리면(❓) 그 지점으로 확인하러** 옵니다(벽도 통과). 조용히 걸으면 들킬 위험이 줄어듭니다. 벽을 사이에 두면 서로 공격할 수 없습니다.
   - 🌦️**날씨·🌡️체온**: 🌧️비=체온↓, 🥵폭염=체온↑로 **서서히** 변합니다. 실내·🚂기차·🌳은신처·옷으로 완화하고, 저체온/고온은 멘탈·체력에 악영향. 상태창에 각 인원의 🌡️체온 표시.
   - 🦠**감염**: 좀비에 맞으면 일정 확률로 감염. **한 번에 죽지 않고 서서히 진행**되며, 진행 중에는 상태창 옆에 💬말풍선 경고("몸이 으슬으슬 춥다" 등)가 가끔 떠 알려줍니다. 발열로 체온이 오르기도 하니 약(💊)으로 일찍 치료하세요(100%면 변이/사망).
   - 🆘생존자 구출, 평야/강에선 **외부 농사** 파종 + 🪵울타리
   - 🛠️**기차 내구도**는 시간이 아니라 **기차에 몰려든 좀비 수에 비례**해서만 감소. 좀비는 평소 기차 쪽으로 서서히 모여들어 붙으면 기차를 공격하고, **처치하거나 가까이서 유인해 떼어내면** 손상이 멈춥니다(상단에 공격 중인 좀비 수·감소율 표시).
   - 💀**오래 정차할수록 강한 좀비가 출현**: 👟질주 좀비(빠름) · 거구 좀비(체력↑·기차 피해↑) · 흉포 좀비(공격력 2배). 초반엔 일반 좀비뿐이지만 시간이 지나며 섞여 나오니, 필요한 것만 챙기고 빨리 떠나는 게 중요합니다.
4. **복귀 → 정비·관리** — 재료/부품으로 수리, 객차 증설, 농사·동료 관리.
   - 한 번 정차해 탐험한 뒤에는 **최소 25km를 더 달려야** 다시 탐험할 수 있습니다(같은 자리 즉시 재약탈 방지). 연료가 0이라 불모지에 갇힌 경우만 예외로 그 자리에서 재수색이 가능합니다.

## 특성 (모든 인원 1개 보유)

**게임 시작 시 본인(차장)의 특성을 직접 선택**합니다. 동료는 다양한 특성을 지닌 채 합류합니다.

탐험형(직접 나가는 본인에게 큰 효과 / 동료가 가지면 정차 중 기차 방어):

| 특성 | 효과 |
|---|---|
| 🧭 생존전문가 | 파밍 성공률 +10%p, 좀비 탐지 범위 감소 |
| 👟 빠른 걸음 | 탐험 이동 속도 +28% |
| 💪 강한 공격력 | 근접 공격 위력·사거리·부채꼴 각도 ↑ |
| 🧱 튼튼함 | 좀비 접촉 피해 감소 |
| 🔑 열쇠공 | 제작전문가 없이도 잠긴 문을 **빠르고 조용히** 땁니다 |
| 🌡️ 안정형 체온 | 체온이 **천천히** 변함 — 추위·더위에 강함 |
| 🧳 짐꾼 | 기본 휴대 무게 한도 +8 |
| 🛠️ 기계전문가 | 파밍지에서 부품으로 자동차에 **알람 설치** → 작동 시 넓은 반경의 좀비를 유인 |

기차 패시브:

| 특성 | 효과 |
|---|---|
| 🌾 농사전문가 | 농작물 수확·성장 ↑, 울타리 보호 ↑ |
| 🔧 제작전문가 | 수리·증설·울타리 비용 ↓, 주행 중 내구도 회복, 자물쇠 따기 |
| 🔬 연구원 | 감염 진행 절반, 연료 효율 ↑ |
| 🪑 가구전문가 | 전원 멘탈 회복 ↑, 외부 농사·울타리 효율 ↑ |

## 무기 (⚔️)

하단 **⚔️ 무기** 메뉴에서 재료·부품으로 무기를 제작합니다. 공격은 클릭한 방향으로 **부채꼴**로 휘둘러지며, 무기는 **사거리(반지름)·각도(휘두르는 폭)·공격력**을 올려줍니다. 단 **내구도**가 있어 적중할 때마다 닳고, 0이 되면 부서져 맨손으로 돌아갑니다. **제작전문가**가 탑승해 있으면 제작 시 **내구도 +40%** 보정을 받습니다.

| 무기 | 사거리 | 각도 | 공격력 | 내구 | 비용 |
|---|---|---|---|---|---|
| ✊ 맨손 | 52 | 28° | 1 | ∞ | - |
| 🏏 몽둥이 | 64 | 60° | 2 | 40 | 재료6·부품1 |
| 🔱 창 | 104 | 30° | 2 | 30 | 재료8·부품2 |
| 🔪 마체테 | 72 | 78° | 3 | 50 | 재료10·부품2 |
| 🪓 도끼 | 66 | 90° | 4 | 26 | 재료12·부품3 |

창은 길고 좁게(찌르기), 도끼·마체테는 짧지만 넓게(휘두르기) 닿습니다.
(💪 강한 공격력 특성은 무기 위에 공격력 +1·사거리 +15%·각도 +10°를 더 얹어줍니다.)

## 지도 (🗺️)

노선 위 **기차의 현재 위치(🚂)**, 다가오는 지형(바이옴 띠), 정차 목표(🎯), 외부 농장 핀(📍)을 한눈에 볼 수 있습니다. 농장이 익을 즈음 미리 정차 목표를 설정해 수확하세요.

## 농사

- **내부 농사**: 인원 1명을 점유(그 사람의 특성 효과 정지)하는 **응급용**. 실시간 성장.
- **외부 농사**: 평야·강 탐험 중 '밭 자리'에 파종(농사/가구/제작 전문가 필요) + 🪵울타리로 보호.
  지도에 📍핀으로 표시되며, 일정 거리 후 성숙. **다시 그 지점에 돌아올 때 정차해 수확**.
  제때 돌아오지 못하면 망가집니다(울타리가 있으면 한 번 버팀).

```
  config.js   # 모든 밸런스 상수 (자원/연료/needs/감염/탐험/농사)

  explore.js  # 탐험 2.5D: 건물 진입 파밍/은신/감염/체온/부채꼴 공격/외부 농사
  ui.js       # 상단 HUD + 항시 크루 카드 + 모달(지도/농사/동료/정비…)
  actions.js  # 운행/수리/증설/영입/치료/농사/수확/정차목표
  game.js     # 메인 루프 · 입력 · 모드 전환 · 초기화
```
