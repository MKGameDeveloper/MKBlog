# 🎣 낚시생활 (Fishing Life)

webfishing에서 영감을 받은 **힐링 낚시 시뮬레이터**. 순수 HTML/CSS/JavaScript로 만들어졌고
별도 서버·빌드·설치 없이 `index.html`만 열면 바로 플레이됩니다.

## 실행 방법
`index.html` 파일을 더블클릭해서 브라우저로 열기만 하면 됩니다. (모바일 화면 비율에 맞춰 디자인됨)
진행 상황은 브라우저 `localStorage`에 자동 저장됩니다.

## 조작법
- **이동**: 방향키 / `WASD`, 또는 화면 좌측 하단 **D-패드**. 캐릭터가 직접 낚시터를 돌아다닙니다.
- **낚시**: `Space`(스페이스바) **전용** — **낚시 시작(던지기) · 챔질 · 끌어올리기(꾹)** 모두 Space, 또는 우측 하단 **행동 버튼**. **물가에 가까이 서면** 머리 위에 `🎣 낚시 (Space)` 안내가 떠요.
- **상호작용**: 📜 의뢰판 · 🪤 통발 · 🏪 상점 · 🏢 부동산 · 🏡 집 · 🪧 이동표지 앞에서 `E` / `Enter` / `Space`(또는 행동 버튼).
  이 대상들엔 항상 머리 위에 **작은 화살표(▼)** 가 떠 있고, 가까이 가면 화살표가 커지며 금색으로 강조됩니다. (E는 낚시엔 동작하지 않아요)

## 플레이 방법
1. 캐릭터를 움직여 **물가 근처**로 가면(정해진 자리 없이 아무 물가에서나) `Space`로 낚싯대를 휘둘러 던집니다.
2. 입질(❗)이 오면 `Space`로 **챔질**(캐릭터가 낚싯대를 챔!).
3. **끌어올리기 미니게임**: `Space`(또는 행동 버튼·화면)를 **꾹 누르면** 초록 칸이 위로 올라가요.
   초록 칸 안에 물고기를 가두면 오른쪽 게이지가 차오르고, 다 차면 포획 성공!
4. 잡은 물고기를 팔려면 **🏙️ 도심지로 이동** → **🏪 상점**의 ‘💰 판매’ 탭에서 판매. (가방에서는 판매 불가, ‘전시’만 가능)
5. 도심지의 **🏪 상점**에서 낚싯대·배·미끼·통발을 사고, **🏢 부동산**에서 **집과 아쿠아리움**을 구매합니다.
6. **📜 의뢰 게시판**(낚시터에 있음)에서 의뢰를 받고 요청한 물고기를 잡아 납품. 게시판은 **30분마다 자동 갱신**돼요.
7. **🪤 통발**: 물가의 통발 표지 앞에서 행동 → 설치, 시간이 지난 뒤 다시 행동 → 수거.
   - 미끼가 없을 땐 걱정 마세요: 낚시터의 **🪱 흙더미**에서 지렁이를 캐거나(시간대당 1회),
     **⚗️ 제작**에서 **🍂 막미끼**를 재료·돈 없이 무료로 만들 수 있어요(효과는 약함). → 게임오버 걱정 없음!
8. **🪧 이동표지**(또는 하단 🗺️ 지도)로 도심지·다른 낚시터·바닷가로 떠납니다.
9. **시간은 저절로 흘러갑니다** — 한 시간대(아침→낮→저녁→밤)가 **약 10분**에 걸쳐 자연스럽게 넘어가요.
   **⏭️ 시간 보내기**를 누르면 그동안 **실제로 흐른 시간만큼**(10분당 한 시간대) 한꺼번에 건너뜁니다.
   (예: 마지막 시간 변화 뒤 25분이 지났다면 두 시간대를 한 번에. 날짜·계절도 함께 넘어가요.)
   낚시 중이거나 메뉴를 열어둔 동안엔 자동 진행이 잠시 멈췄다가 이어집니다.

## 구현된 핵심 시스템
| 시스템 | 설명 |
|---|---|
| 🎣 **낚시 미니게임** | 초록 칸에 물고기를 가두는 방식. 장비가 좋을수록 쉬워짐 |
| 🏙️ **도심지** | 걸어서 가는 번화가 — 🏪 상점(판매·구매)과 🏢 부동산(집 구매)이 있음 |
| 🐟 **41종 물고기** | 흔함~전설 5등급. 잡동사니(장화·깡통·보물상자)도 등장 |
| 🗺️ **7개 낚시 환경** | 연못·강·계곡·호수·바다·동굴호수·달빛해변 — 각기 다른 어종 |
| 🌦️ **계절·날씨·시간** | 봄/여름/가을/겨울, 맑음/흐림/비/눈, 아침/낮/저녁/밤마다 출현 어종 변화 |
| 🪱 **미끼 제작** | 재료를 모아 떡밥·제물낚시·발광미끼·월광미끼 등 제조. 유인력이 높을수록 입질이 빠르고 **상위 어종 확률↑** |
| 🎣 **낚싯대 강화** | 등급이 높을수록 **찌(초록 칸)가 더 크고 더 빠르게** 움직임. 또한 낚싯대가 물고기 등급을 앞설수록 **하위 어종의 활동성이 약해져** 쉽게 잡힘(전설 등 상위어는 그대로 활발) |
| 📜 **의뢰(퀘스트)** | NPC가 요청한 물고기를 납품하고 골드·재료 보상 획득 |
| 🪤 **통발** | 물에 설치 후 시간이 지나면 물고기가 들어옴 → 수거 |
| 🎣 **낚싯대 5종 / ⛵ 배 4종** | 장비 업그레이드로 더 좋은 지역·어종 도전 |
| 🏡 **집 & 아쿠아리움** | 집을 사고 어항을 설치해 잡은 물고기를 전시·수집 |
| 📖 **도감** | 잡은 물고기 기록 및 최대 크기 수집 |

## 진행 흐름 (대략)
연못·강에서 시작 → 잉어·붕어 팔아 미끼/낚싯대 구입 → 계곡(300G) 해금 →
보트(800G)로 호수 → 모터보트로 바다·동굴 → 집과 아쿠아리움 →
전설 장비로 달빛해변의 리바이어던·크라켄 도전.

## 파일 구조
```
index.html        화면 레이아웃 + 캔버스 + 화면 조작 버튼
css/style.css     스타일 (아늑한 힐링 테마)
js/data.js        물고기·지역·장비·미끼·의뢰 데이터
js/fishart.js     어종별 외형을 닮은 캔버스 그림 렌더러 (이모지 대신 진짜 물고기 그림)
js/state.js       게임 상태 + 저장/불러오기(localStorage)
js/world.js       돌아다니는 캐릭터 오버월드 (지형·이동·상호작용·렌더링)
js/fishing.js     낚시 미니게임 (월드 위에 오버레이로 표시)
js/actions.js     상점·판매·제작·의뢰·통발 등 버튼 로직
js/ui.js          모든 패널 UI 렌더링
js/game.js        시간/날씨/계절 루프, 포획 처리, 초기화
```

처음부터 다시 시작하려면 우측 상단 **↺** 버튼을 누르세요.
