# 🌾 Farm Today

스타듀밸리를 레퍼런스로 삼은 **2D 픽셀 농장 웹게임**입니다.
순수 HTML5 Canvas + 바닐라 JavaScript로 만들어졌고, **빌드 도구·외부 라이브러리·이미지 에셋이 전혀 없습니다.**
모든 픽셀 아트는 코드로 직접 그립니다.

## 실행 방법

### 방법 1) 그냥 열기 (가장 간단)
`index.html` 파일을 더블클릭해서 브라우저로 엽니다.

### 방법 2) 로컬 서버 (저장 기능이 더 안정적)
일부 브라우저는 `file://`에서 저장(localStorage)을 제한합니다. 그럴 땐 서버로 여세요:

```bash
# 이 폴더에서
python -m http.server 8000
# 그 다음 브라우저에서 http://localhost:8000 접속
```

## 조작법

| 키 | 동작 |
|---|---|
| `WASD` | 이동 (방향키는 사용하지 않음) |
| 마우스 위치 | 캐릭터 기준으로 **4방향 조준** (커서 쪽을 바라봄) |
| 마우스 클릭 | 조준한 방향으로 **도구 휘두르기**. 괭이/물뿌리개는 **길게 눌러 차징**(범위↑). 차징 중엔 **절반 속도로 이동 가능**, 휘두르는 순간엔 정지 |
| `1`~`9`, `[`, `]` | 핫바 선택 (도구 6종 + 씨앗/설치물) |
| `E` | 상호작용 (수확·기계 수거·상점 카운터·침대·출하상자·광산·산출물) / 문(매트)은 걸어 들어가면 입장 |
| `I` | 가방 열기 · `B` 건설 메뉴 · `M` 지도(내 위치·주민·시설) |

**도구**: 괭이(밭갈기) · 물뿌리개(물) · 망치(돌 채굴) · 낫(잡초/수확) · 도끼(나무) · 검(동굴 전투)

## 구현된 시스템

- **분리된 맵(농장 ↔ 마을 ↔ 산)**: 스타듀처럼 별개의 맵. 지도/메뉴 이동이 아니라 **길 끝으로 걸어 나가면** 인접 맵이 로드됨 (농장 동쪽 ↔ 마을 ↔ 동쪽 산). 광산은 마을이 아니라 **별도의 산 맵**에 있음
- **건물 내부 출입**: 문(매트)으로 **걸어 들어가면 건물 내부 맵**으로 전환되고, 출구로 나가면 원래 자리로 복귀
- **상점 영업/근무**: 상점 주인이 **영업시간(오전 9시~오후 5시)** 에 카운터에 출근해 있을 때만 카운터에서 구매 가능. 영업 종료 시엔 주인이 퇴근해 이용 불가
- **건물 아트**: 박공 지붕·기와 결·돌 기초·틀 창문·아치문·굴뚝·도머창·넝쿨. 벽 재질도 **하프팀버(튜더)·통나무·판자·돌** 4종으로 건물마다 다름
- **마을(돌 광장)**: 스타듀밸리 마을처럼 **중앙 돌바닥 광장**을 건물들이 둘러싸고, **분수·가로등·울타리**와 둘레를 감싼 **숲**, 꽃·덤불 등 자연물로 구성
- **계절 단풍**: 나무가 봄(꽃)·여름(초록)·가을(단풍)·겨울(앙상+눈)에 따라 다르게 보임
- **NPC(마을 주민) + 대화**: 마을을 배회하는 주민들. 바라보고 **E**로 대화하면 **부드러운 2D 일러스트 초상화**(픽셀 아님)와 대사가 나옴 — 나이·직업에 맞춘 외형(예: 촌장은 노인). 상점 주인 피에르는 영업시간에 상점에서 근무
- **지도(`M`)**: 농장·마을 전체 도식에서 **내 위치(빨간 ●)·주민 위치·주요 시설(집/상점/광산/출하/문)** 을 실시간으로 확인. 지도에서 수동 저장도 가능
- **농사**: 갈기 → 물주기 → 심기 → (물 준 날만) 성장 → 수확
- **작물 특성**: 작물마다 성장 일수 / 1회성·다회 수확(토마토·포도 등) / 수확 개수 다름
- **계절**: 봄·여름·가을·겨울, 한 계절 28일. 계절별 상점 씨앗이 다르고 제철 아니면 시듦
- **시간**: 스타듀처럼 10분 단위 `hh:mm` 시계 / 침대에서 자면 다음 날 (작물 성장·산출물 갱신)
- **렌더링**: 기기 해상도(devicePixelRatio)에서 정수배 확대 — 픽셀은 또렷하고 텍스트는 고해상도로 선명
- **행동/체력**: 도구질에 휘두르기 애니메이션, 그동안 이동 잠금, 행동마다 체력(에너지) 소모
- **수확/수거는 상호작용(E)**: 다 자란 작물·발효통 완성품(와인 등)·축산물을 모두 **E**로 거둠 (낫은 잡초·시든작물 정리용)
- **물뿌리개 물 제한**: 물뿌리개에 물이 한정되어, 다 쓰면 **연못에 대고 보충**해야 함 (강화 시 용량↑, 아침엔 가득)
- **스프링클러**(건설 B에서 광물로 제작): 밭에 설치하면 **매일 아침 자동 급수**. 스타듀밸리처럼 3단계 — 기본(상하좌우 4칸)·품질(3×3 8칸)·이리듐(5×5 24칸)
- **도구 차징/강화**: 괭이·물뿌리개는 Space를 **길게 눌러 차징**하면 한 번에 여러 칸 처리. **강화 단계가 높을수록 더 넓은 범위**(1칸→3칸→3×3)
- **채집**: 망치로 돌 캐기(큰 돌은 망치 업그레이드 필요), 낫으로 잡초, 도끼로 나무
- **2차 가공**: 발효통(과일→와인/채소→주스), 제분기(밀→밀가루), 버터 교반기(우유→버터), 보존 단지(→잼)
- **건물**: 닭장(달걀)·외양간(우유)·마구간(이동속도↑) 구매·설치
- **상점**: 씨앗 구매 / 농작물·가공품·광물 판매 / 망치 업그레이드
- **동굴(산의 광산)**: 산 맵의 광산 입구(E)로 진입. 깊이별 광물 채굴(구리·철·금·석탄·보석), 슬라임 처치, 사다리로 더 깊이
- **저장/로드**: 브라우저 localStorage. 잠을 자면 자동 저장, 지도(`M`)에서 수동 저장

## 파일 구조

```
index.html        진입점 (스크립트 로드 순서)
style.css         캔버스 스케일링(픽셀 퍼펙트)
js/
  data.js         작물·아이템·가공레시피·건물·도구 데이터 (← 여기서 콘텐츠 추가)
  sprites.js      코드로 그리는 픽셀 아트
  input.js        키보드/마우스
  world.js        농장 맵·타일·오브젝트·농사 로직
  player.js       플레이어 이동·애니메이션·인벤토리·핫바
  cave.js         동굴 채굴·전투
  ui.js           HUD·상점·건설·가방 메뉴·알림
  game.js         메인 루프·시간·도구사용·거래·저장
test/headless.js  Canvas/DOM 스텁으로 게임 로직을 구동하는 검증 (node test/headless.js)
```

## 콘텐츠 추가하기

데이터 기반 설계라 `js/data.js`에 항목만 추가하면 게임에 바로 반영됩니다.
예) 새 작물:

```js
melon2: { name:'수박', seasons:['summer'], grow:12, regrow:null,
          yieldMin:1, yieldMax:1, fruit:true, sell:90, seed:60, tint:'#3a9a4a' },
```

## 알려진 한계 (프로토타입)

- 픽셀 아트는 코드로 그린 단순한 형태(스프라이트 시트 미사용)
- NPC/대화/퀘스트, 낚시, 요리, 결혼 등은 미구현
- 동물은 시각적 표현 위주(산출물은 건물에서 매일 수거)
