참고용 레퍼런스 폴더 (게임에 번들되지 않음)
=========================================

여기에 그림체 레퍼런스 이미지를 넣어줘. 내가 painter.dart로 그릴 때 이 폴더를
열어보고 비율·팔레트·실루엣·음영을 최대한 비슷하게 코드로 반영함.
(레퍼런스를 픽셀 단위로 복제하는 게 아니라, 스타일을 참고해 오리지널 도형 코드로
새로 그리는 것)

폴더
  character/  스컬(플레이어가 장착하는 직업 해골) 레퍼런스
  monster/    일반 몬스터 레퍼런스
  boss/       보스 레퍼런스
  map/        배경/타일/지형/소품 등 맵 레퍼런스

파일 이름 팁 (이렇게 지으면 어떤 게임 요소인지 바로 매칭돼서 좋아)
  character/skull_warrior.png, character/skull_mage.png ...
  monster/goblin.png, monster/dwarf.png, monster/dark_elf.png ...
  boss/yggdrasil.png, boss/knight.png ...
  map/forest_bg.png, map/ground_tile.png, map/props.png ...
  여러 종류를 한 장에 모은 시트여도 됨 — 그 안에서 골라 참고할게.

진행
  레퍼런스를 다 넣은 뒤 "다 넣었어"라고 알려주면,
  스컬 → 몬스터 → 보스 → 맵 순서로 다시 그릴게.
